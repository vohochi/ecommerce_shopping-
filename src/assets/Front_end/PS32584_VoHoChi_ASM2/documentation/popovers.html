Popovers

Documentation and examples for adding Bootstrap popovers, like those found in iOS, to any element on your site.

Bootstrap Popovers documentation

Example

Four options are available: top, right, bottom, and left aligned. Use the focus trigger to dismiss popovers on the user’s next click of a different element than the toggle element.
